# PenguFamily — Roadmap

> A living document of planned features and improvements for future updates.

---

## ✅ Foundation (Current)

- [x] Egg hatching + onboarding (choose archetype & name)
- [x] 5 archetypes: Windblade, Shadow, Starlight, Chaos, Royal
- [x] Stat system: hunger, happiness, energy + mood computation
- [x] MoodEngine: stat decay over time, offline time processing
- [x] PenguAI: wandering, mouse reactions, POI visits, idle chatter
- [x] Drag & drop with personality reactions + fling physics
- [x] Fast-drag detection with personality dialogue
- [x] Sleep system (lullaby item, 30 min sleep)
- [x] Sprite rendering with archetype/stage-based sprites
- [x] Chat bubbles (personality-based dialogue system)
- [x] Shop with 6 items (snack, treat, energy drink, toy, xp boost, lullaby)
- [x] Quest system: daily, weekly, repeatable
- [x] Game tracking: wins/losses, XP, coins, win streaks
- [x] Champ select reactions (picks, bans, suggestions)
- [x] Lobby join reactions
- [x] Hub modal with Status, Shop, Quests, Settings tabs
- [x] Settings: overlay toggle, reset progression
- [x] Evolution stages: baby → adult → elder

---

## 🔜 — Cosmetics & Personalization

- [ ] **Cosmetic shop items** — hats, accessories, trails that visually modify the pengu sprite
- [ ] **Equip system** — equip/unequip cosmetics from the Status tab
- [ ] **Rare cosmetics** — special items earned from achievements or long win streaks
- [ ] **Custom chat bubble styles** — archetype-specific bubble skins purchasable in shop
- [ ] **Pengu title system** — earn titles based on behavior ("Speedster", "Sleepyhead", "Undefeated")

## 🎮 — Mini-Games & Activities

- [ ] **Pengu mini-games** — small idle games (fishing, treasure hunt) during queue times
- [ ] **Daily puzzle** — a small daily challenge for bonus coins/XP

---

## 🧠 — Advanced AI & Behavior

- [ ] **Relationship level** — pengu remembers how often you interact, unlocks new dialogues
- [ ] **Memory system** — pengu references past games ("Remember that 20/0 game?")
- [ ] **Mood influence on gameplay** — happy pengu gives luck messages, sad pengu is more withdrawn
- [ ] **Context-aware dialogue** — comments on specific champions, game modes, time of day
- [ ] **Nightmare mode** — if neglected too long, pengu becomes "corrupted" (reversible)

---

## 🎨 — Visual Polish

- [ ] **Animated sprites** — frame-by-frame walk, idle, sleep, eat, celebrate animations
- [ ] **Particle effects** — sparkles on level up, hearts on pet, food particles on feed
- [ ] **Smooth transitions** — ease-in/out on all movements, bounce on landing after fling
- [ ] **Weather effects** — pengu reacts to in-client "weather" (rain when losing streak, sun on wins)
- [ ] **Night mode** — pengu gets sleepy at night, different ambient behavior
- [ ] **Stage-specific visuals** — baby/adult/elder have distinct idle behaviors and sizes

---

## 🏆 — Achievements & Progression

- [ ] **Achievement system** — permanent unlockable achievements with badges
  - "First Win", "100 Games", "Max Level", "All Archetypes Hatched"
- [ ] **Prestige system** — reset pengu at max level for prestige points and exclusive rewards
- [ ] **Collection book** — catalogue of all items, cosmetics, achievements earned
- [ ] **Season pass** — time-limited quest chain with exclusive cosmetic rewards
- [ ] **Milestone rewards** — bonus coins/items at game count thresholds (10, 50, 100, 500)

---

## 🐣 — Multi-Pengu & Breeding

- [ ] **Multiple pengus** — hatch and manage a family of pengus
- [ ] **Pengu switching** — swap active pengu from the hub
- [ ] **Breeding** — combine two pengus to create a hybrid archetype
- [ ] **Nursery** — inactive pengus rest in a nursery, slowly gaining energy
- [ ] **Pengu personalities evolve** — traits shift slightly based on player behavior over time

---

## 🔧 Ongoing / Quality of Life

- [ ] **Sound effects** — optional SFX for interactions, level up, evolution
- [ ] **Notification system** — gentle alerts when pengu is hungry/sad/quest complete
- [ ] **Performance optimization** — reduce DOM operations, optimize sprite rendering
- [ ] **Localization** — multi-language support (EN, TR, etc.)
